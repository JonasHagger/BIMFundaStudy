Concept cards

VR vs AR (+ 2 systems each)

CompareExamCard 1

Exam: DE-Q62 — difference + 2 systems each · MP-Q22 (T/F plans & VR/AR)

VR a fully virtual environment that replaces/mimics the real world. Systems: CAVE, HMD (Powerwall).

AR virtual objects added onto the real world (overlay). Systems: DEPLANAR, HoloLens/TwinBIM (Google Maps Live View).

Exam answerVR replaces reality (immersive virtual world — CAVE, HMD); AR overlays virtual objects on the real world (DEPLANAR, HoloLens).

Rendering

DefinitionExamCard 2

Exam: DE-Q64 — what is rendering?

Generating a photorealistic or non-photorealistic image from a geometric model, accounting for lighting, surfaces, and visualization parameters. The IFC model has geometry but little visualization info → needs a scene description + rendering technique.

Techniques Phong shading · Raytracing · Global Illumination · Radiosity

Modes offline (still/animation) vs realtime (game engine, VR/AR)

Exam answerRendering = producing a 2D image from a 3D scene (lighting, materials, camera).

Animation

DefinitionExamCard 3

Exam: DE-Q64 — what is animation?

(Lat. animare, "to give life to.") Any technique that generates movement from single static images; the viewer follows a predefined, pre-calculated path (flip-book principle).

Keyframe transition defined by start/end frames

Interpolation the in-between frames ("tweening")

Exam answerAnimation = movement generated from static frames along a pre-calculated path (keyframes + interpolation). Tool: Revit Walkthrough.
