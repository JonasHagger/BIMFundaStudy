What is parametric modeling?

DefinitionCard 1

Foundational — the idea behind the constraint cards.

Shifts from fixed-shape to flexible, rule-based models by defining parameters, constraints and dependencies, so the entire model updates consistently. Parameters = dimensions + geometric attributes (shape/location).

Parametric BIM modelers & families

ConceptCard 5

Foundational — how object-based + parametric geometry combine.

Combine object-based modeling + parametric geometry: user-defined reference axes/planes (grids, floor levels), user-defined object types ("families"), and predefined constraints (alignment, orthogonal, parallel, coincidence, equal distance). A family = a class defined by relations & rules controlling the parameters by which instances are generated — no programming needed (Eastman 2008). Extend via Revit + Dynamo, Grasshopper, Allplan Visual Scripting.
