How solids are represented: the explicit way (boundary representation — faces/edges/vertices) vs the implicit way (procedural construction history — sweeps + CSG booleans), plus free-form curves. This is the most sketch-heavy chapter.

Solid modeling→Explicit = Boundary Representation/Implicit = Procedural (construction history)
Implicit (CSG / procedural)Explicit (B-rep)

Exam answerExplicit describes an object by its boundaries; implicit describes it by its procedural construction history. Implicit → easy to modify, small files, but must be interpreted; explicit → easy to interpret, but hard to modify and larger. (Triangle meshes → clash detection/QTO; procedural → modification.)

B-rep face sets: Triangulated vs Indexed

CompareExamCard 4

Exam: DE-Q46 (which is more memory-efficient) · DE-Q48 (problem with triangulated surfaces)

Triangulated Face SetIndexed Face Set
each triangle stores its 3 vertices → vertices repeated (redundancy)vertices stored once, each gets an ID; faces reference indices
simple; STL, PLY, OBJ, glTFmore memory-efficient; OBJ/PLY/X3D/IFC "IndexedFaceSet"

Problems with the triangulated surface approach

Only planar faces; curved surfaces → many triangles; vertices stored multiple times; neighbourhood relationships not stored → gaps & overlaps hard to detect.

Exam answerIndexed is more memory-efficient because each vertex is stored only once and referenced by index (triangulated repeats shared vertices).

Exam answerTetra: V=4, E=6, F=4. Every edge is bounded by 2 vertices; every triangular face by 3 edges. (Euler: V−E+F = 4−6+4 = 2. ✓)

Two overlapping squares A (left) & B (right)

Exam answerRead the result: merged = Union, overlap-only = Intersection, A with B bitten out = Difference. CSG = primitives combined by booleans in a tree.

Free-form curves: Bézier → B-Spline → NURBS

SketchExamCard 7

Exam: DE-Q49 (surface-modeling curves) · MP-Q21 (explain/draw Bézier)

Bézier: passes through first & last control point, approximates the rest

Key idea
Bézierdegree = n−1 control points; Bernstein basis; moving 1 point → global change; more points → higher degree → oscillations
B-Splinejoins segments; local control; degree independent of #control points (smoothness of low degree + many points)
NURBSB-Spline + weight per control point (extra coordinate) → exact circles/ellipses/conics
Exam answerThree curves: Bézier, B-Spline, NURBS. B-Splines add local control + decoupled degree; NURBS add weights to represent conic sections (circles) that B-Splines can't.
